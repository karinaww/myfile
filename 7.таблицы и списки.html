2.Нумерованный список

Пошаговая инструкция для регистрации на GitHub:

  1. Перейдите на сайт GitHub
  2. Нажмите на кнопку "Sign up"
    На главной странице сайта найдите и кликните на кнопку "Sign up" (Зарегистрироваться).
  3. Заполните форму регистрации
    Введите свой адрес электронной почты, придумайте уникальное имя пользователя и надёжный пароль. Следуйте указаниям на экране.
  4. Подтвердите свою электронную почту
    GitHub отправит на указанный вами email письмо с кодом подтверждения. Откройте его и следуйте инструкциям для завершения регистрации.
  5. Настройте ваш аккаунт
    После подтверждения почты выберите подходящий тарифный план (можно начать с бесплатного) и настройте свой профиль, добавив фото и описание.

Теперь ваш аккаунт на GitHub готов к использованию!
